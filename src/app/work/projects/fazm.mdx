## Overview

Fazm is a voice-first AI agent that controls your entire Mac. It's a computer agent — not a chatbot. Users issue voice commands, and Fazm executes tasks by controlling the browser, manipulating documents, managing emails, and operating applications. Features DOM-level browser control, a local memory layer that builds a personal knowledge graph, and multi-app integration with Gmail, Calendar, Slack, Jira, VS Code, Figma, and more. Open-source and free to download.
