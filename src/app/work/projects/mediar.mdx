## Overview

Mediar is an AI-native automation platform that replaces traditional RPA tools like UiPath. It automates repetitive computer tasks — data entry in legacy software like SAP, ERP systems, and healthcare platforms — using OS-level accessibility APIs instead of fragile pixel/vision-based approaches, achieving 99.9% reliability. Users record a workflow once and deploy it anywhere, with AI that self-heals when interfaces change. Co-founded with Louis Beaumont, Mediar raised a $2.8M seed round backed by LG Technology Ventures, Founders Inc, Seed Club Ventures, and Top Harvest Capital.
